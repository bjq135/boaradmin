## BoarAdmin(beta版本)

### 简介
BoarAdmin 是一个基于 Vue.js 和 ElementPlus 开发的适配 BoarCMS 的后台管理系统

### 安装

```sh
npm install
npm run dev
```

### 特性

- 登录模块
- 文章管理模块 (开发中)
- 分类管理模块 (待开发)
- 页面管理模块 (待开发)
- 标签管理模块 (待开发)
- 用户管理模块 (待开发)
- 日志管理模块 (待开发)
- 附件管理模块 (待开发)
- 验证码管理模块 (待开发)
